# Snake Game with path finding

## Description

This is a snake game using A* algorthim to find an efficient path

## Setup

### Requirements

- Python 3
- pygame

### Installation

1. Clone the repository.
2. Install the dependencies:

```sh
pip install pygame
```

### Running the Application

To run the application:

```sh
python SnakeGame/main.py
```
